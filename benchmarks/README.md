# Benchmarks

Harness and raw results for FRIDAY's serving performance.

Planned measurements, run locally and committed with raw output:

- tokens/sec, single request and batched
- end-to-end latency (p50/p95/p99) from query to first audio
- peak memory
- comparison against a llama.cpp baseline on the same model and quantization

Nothing is published here that was not measured on real hardware, with the raw output committed alongside the summary table.
